# Bex Project Catalogue

A responsive project catalogue for Bex digital products, completed applications, dashboards, automation tools, and active development projects.

## Overview

The Bex Project Catalogue provides a central place to showcase Bex projects with screenshots, project descriptions, technology stacks, GitHub repository links, live application links, and project status indicators.

The catalogue is built as a lightweight static website using structured project data from `projects.json`.

## Features

- Responsive project catalogue layout
- Search by project name, category, technology, or status
- Sort by recommended order, name, category, or status
- Project screenshot preview modal
- GitHub and live application links
- Completed and in-progress project status badges
- Structured reusable project data through `projects.json`
- Mobile-friendly layout

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- JSON
- Vercel-ready static deployment
